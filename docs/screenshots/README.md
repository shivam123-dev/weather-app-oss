# Screenshots

Add UI screenshots to display in the main README.

Suggested captures:
- `light.png`: Light theme on clear weather
- `dark.png`: Dark theme on clouds
- `rain.png`: Rain background
- `clear.png`: Clear sky background

How to capture:
1. Open `docs/index.html` in your browser.
2. Enter a city (e.g., "London").
3. Toggle theme via the moon/sun button.
4. Use your OS screenshot tool; save into this folder with the filenames above.

Tip: For consistent sizes, capture at ~1280720.
